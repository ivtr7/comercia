# comercIA

## Soluções em Inteligência Artificial

Sistema de automação de atendimento e qualificação de leads via WhatsApp e Instagram.

## Tecnologias

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como executar localmente

Requisitos: Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Passo 1: Instalar dependências
npm i

# Passo 2: Iniciar o servidor de desenvolvimento
npm run dev
```

## Como fazer o build

```sh
npm run build
```
